# 打标签辅助工具

## 文件结构
src/multi-player.html -- 多视频播放器
src/tag-player.html -- 单视频打标签播放器
src/check-player.html -- 校验界面播放器

## 使用方法
双击html文件，用默认浏览器打开。
暂停时，点击+，添加时刻，点击2次：第1次是场景结束时刻；第2次是下一个场景的开始时刻。
若发生错误，请手动更改。
